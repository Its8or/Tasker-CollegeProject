TASKER

Porque fizemos o Tasker?

Apesar de já haverem sites e até mesmo programas de gerenciamento de tarefas, quisemos fazer o Tasker para mostrar nossos conhecimentos em programação.

Como Usar

Usar o nosso Task Manager é simples e intuitivo. Siga estes passos básicos para começar:

  1. Use o botão para ser redirecionado.


    2. Ou use a Aba TASKS no topo da página à direita.

  2. Crie uma tarefa.

    3. Na tela inicial, insira os dados nos campos desejados e confirme a atividade.

  3. Organize suas tarefas.

    4. Defina datas limite para suas tarefas.

    Acompanhe o progresso: Marque as tarefas como "em andamento" ou "concluídas" conforme avança.

  4. Filtre o que for importante.

    5. Use os filtros para visualizar as tarefas como:

    • Não iniciadas;
    • Em andamento;
    • Concluídas;
