### Regras de nomear galhos(Branches):

#### 1.Faça uma branch com um nome memorável e simples:
* **ex:** 'small-changes' ou 'b_SeuNome';

#### 2. Use um nome exclusivo:
* **ex:** 'b_8or';

#### 3. Nao use pontos ou caracteres especiais:
* **ex:** '8or.2001' ou '8or#branch'
